# 🐾 Axis APP for Therians

Aplicación web desarrollada como proyecto colaborativo orientada a la comunidad therian. La plataforma permite la interacción entre usuarios mediante perfiles, navegación dinámica y formularios interactivos, ofreciendo una experiencia similar a una aplicación real.

## 🚀 Características

- Creación y visualización de perfiles.
- Navegación dinámica entre secciones.
- Formularios con validación de datos.
- Interfaz responsive para dispositivos móviles y escritorio.
- Renderizado dinámico de contenido.
- Manejo de eventos del DOM.
- Integración con APIs para obtención y gestión de datos.

## 🛠 Tecnologías utilizadas

- JavaScript (ES6+)
- HTML5
- CSS3
- Git
- GitHub

## 📱 Diseño Responsive

La aplicación fue desarrollada siguiendo principios responsive para garantizar una experiencia consistente en:

- Computadoras de escritorio
- Tablets
- Dispositivos móviles

## 🎯 Objetivos del proyecto

- Aplicar conceptos de desarrollo frontend.
- Implementar interacción dinámica con JavaScript.
- Trabajar en equipo mediante Git y GitHub.
- Simular funcionalidades presentes en aplicaciones reales.
